MediaPro 303, Fall Semester 2022

About the class:

In this class, the goal was to be able to teach us how to code and use other websites, such as WordPress and Squarespace, to be able to make our own sites. When it came to coding, we learned the basics of HTML, and then went into learning about CSS. HTML creates the basics of a webpage, with things like text, headers, lists, links, adding images, and general things like that. CSS has to do with the stylization of it and being able to change text size, fonts, colors for everything, adding boxes, and anything you can think of for making your site pretty.

Before we started any actual coding, we went over basics of what the Internet is and how it works. We learned what HTML is, how it's used, and how it's helpful. We then got into the actal practic of HTML coding. We worked on that for a few weeks before moving onto CSS. In the midst of all of this, we practiced our coding on a website called FreeCodeCamp, and this helped us to learn new codes while refreshing our memory on codes we learned in class.

After we learned HTML and CSS, we moved onto using outside sites such as WordPress and Squarespace. In this website, you will see my journey with HTML, CSS, WordPress, and Squarespace. As you navigate through each tab, it'll give you more information on what we did in class and what I made.
